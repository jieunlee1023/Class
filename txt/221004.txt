자바에서 제공되는 자료구조 구현 클래스들 
- 컬렉션 프레임 워크

* 프로그램 구현에 필요한 자료구조 (Data Structure)를 구현해 놓은 JDK 라이브러리
* java.util 패크지에 구현되어 있음
* 개발에 소요되는 시간을 절약하면서 최적화된 알고리즘을 사용할 수 있음
* 여러 구현 클래스와 인터페이스의 활용에 대한 이해가 필요함

set 계열 (주머니)
순서가 없고 중복이 불가함

List 계열 
index
순서가 있고 중복 가능

Map 계열
key와 value 구조로 한쌍


 연결 리스트 (LinkedList)
: 선형으로 자료를 관리, 자료가 추가될 때마다 메모리를 할당 받고, 자료는 링크로 연결됨. 
자료의 물리적 위치와 논리적 위치가 다를 수 있음
추가나 삭제할 때 ArrayList보다 용이하다.

● 스택 (Stack) : 가장 나중에 입력 된 자료가 가장 먼저 출력되는 자료 구조 (Last In First Out)

● 큐 (Queue) : 가장 먼저 입력 된 자료가 가장 먼저 출력되는 자료 구조 (First In First Out)

● 트리 (Tree) : 부모 노드와 자식 노드간의 연결로 이루어진 자료 구조 (비선형 자료구조)
