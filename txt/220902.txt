* 배열!
연관된 데이터를 통으로 모아서 관리하기 위해 사용하는 데이터 타입
변수가 하나의 데이터를 저장하는 것이라면,
배열은 여러 개의 데이터를 하나의 변수에 저장하기 위한 것
-동일한 자료형의 순차적 자료 구조(데이터를 어떠한 알고리즘 방식으로 효과적으로 관리하는가)
-인덱스 연산자[]를 이용하여 빠른 참조(접근하여) 가능
-물리적 위치와 논리적 위치가 동일
-배열의 순서는 0부터 시작
- 자바에서는 객체 배열을 구현한 ArrayList 를 많이 활용함

배열 선언과 초기화
배열도 객체임 (new는 메모리에 올릴 때!)
int[] arr1 = new int[10];
arr1에 담기는 값은 주소값임!
int arr2[] = new int[10];
배열은 선언과 동시에 자료형에 따라 초기화 됨 (정수는 0, 실수는 0.0, 객체는 null)
필요에 따라 초기값 지정 가능


모든 프로그램에서 인덱스의 시작은 0부터 시작한다.
