* 조건문 
: 주어진 조건에 따라 다른 실행이 이루어 지도록 구현
'재산이 5천만원 이상이면 10만원을 세금으로 낸다, 그렇지 않으면 내지 않는다'
만약에 (if) (재산이 5천만원 이상이면 =조건문) { 
	세금을 10만원 낸다
}
그렇지 않으면 {
	세금을 내지 않는다
}

★ 조건식은 피라미드 모양으로 범위를 지정한다.

if문 (만약에 ... 라면)
if (조건문) { 
	수행문;
}
조건식이 참인경우 -> 수행문 실행 o
조건식이 참이 아닌 경우 -> 수행문 실행 x

if - else문
if (조건식) {
	수행문 1; : 조건식이 참인경우 수행됨
} 
else {
	수행문 2; : 조건식이 참이 아닌 경우 실행됨
}

if - else if -else 문 
: 조건식이 여러개인경우
if (조건식1) {
	수행문1;
}
else if {
	수행문 2; : 
}
else if {
	수행문 3; : 
}




*반복문 (for문)
for (초기화식;조건식;증감식){
	수행문;
}

변수라는 것을 스스로 생각해서 만들어 활용하고
실행에 흐름을 제어하고 (if) 반복적인 패턴이 보인다면 for문을 활용할 수 있는가?
